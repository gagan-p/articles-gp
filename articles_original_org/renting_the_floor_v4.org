* Core Thesis

#+begin_export html
<div class="thesis">
<div class="thesis-title">
Core Thesis
</div>
<div class="paragraph">
Applications are rented. Foundations are owned.

Operating systems, databases, distributed storage, messaging systems, language runtimes and compilers form 
the foundation upon which modern software is built.

India built one of the world's largest software industries but has only a limited presence in globally adopted
foundational software. This essay argues that this imbalance is strategically significant and that India must 
increasingly own, rather than rent, the software floor.
</div>
</div>
#+end_export

* What Is Foundational Software?

Foundational software provides execution environments or shared
infrastructure upon which other software depends. 

#+begin_export markdown

** The Crucial Difference

The algorithm is the same everywhere. The inputs differ.

| Condition | China | India |
|-----------+-------+-------|
| Can you buy/rent it? | No (sanctions, delete-IOE) | Yes (open market) |
| Is it core? | Yes (state banks, platforms) | No (services firms) |
| Does it differentiate? | Yes (Singles' Day scale) | No |
| Is building cheaper? | Yes (no alternative) | No (renting is cheaper) |
| Strategic independence asked? | Yes (state policy) | No (market logic) |

**Result:** China built. India rented. The remainder of this essay explains why this matters.
