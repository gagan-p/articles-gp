* Core Thesis

#+begin_export html
<div class="thesis">
<div class="thesis-title">
Core Thesis
</div>
<div class="paragraph">
Applications are rented. Foundations are owned.

Operating systems, databases, distributed storage, messaging systems, language runtimes and compilers form 
the foundation upon which modern software is built.

India built one of the world's largest software industries but has only a limited presence in globally adopted
foundational software. This essay argues that this imbalance is strategically significant and that India must 
increasingly own, rather than rent, the software floor.
</div>
</div>
#+end_export

* Executive Summary

#+begin_export html
<div class="summary">
<div class="summary-title">
Executive Summary
</div>
<ol>
<li>India has one of the world's largest software industries.</li>
<li>Its presence in globally adopted foundational software is comparatively small.</li>
<li>Foundational software enables other software and compounds value across the software ecosystem.</li>
<li>This essay compares India with representative global and Chinese projects.</li>
<li>It studies how successful foundational software originates and how it evolves.</li>
<li>It distinguishes a project's origin from its eventual outcome.</li>
<li>It proposes that engineering constraints create foundational software, while institutional custody determines whether it survives.</li>
<li>The resulting framework is applied to India and China.</li>
<li>The framework makes predictions that can be tested against future projects.</li>
</ol>
