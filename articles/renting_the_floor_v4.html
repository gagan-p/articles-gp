Core Thesis

Core Thesis

Applications are rented. Foundations are owned.

Operating systems, databases, distributed storage, messaging systems, language runtimes and compilers form the foundation upon which modern software is built.

India built one of the world’s largest software industries but has only a limited presence in globally adopted foundational software. This essay argues that this imbalance is strategically significant and that India must increasingly own, rather than rent, the software floor.

Executive Summary

Executive Summary

  1. India has one of the world’s largest software industries.
  2. Its presence in globally adopted foundational software is comparatively small.
  3. Foundational software enables other software and compounds value across the software ecosystem.
  4. This essay compares India with representative global and Chinese projects.
  5. It studies how successful foundational software originates and how it evolves.
  6. It distinguishes a project’s origin from its eventual outcome.
  7. It proposes that engineering constraints create foundational software, while institutional custody determines whether it survives.
  8. The resulting framework is applied to India and China.
  9. The framework makes predictions that can be tested against future projects.

The Paradox

India's software inputs predict a mature foundational ecosystem. The observed output does not match.

What Is Foundational Software?

Foundational software provides execution environments or shared infrastructure upon which other software depends.

Definition. Software is foundational when its removal forces a rewrite of everything built on it.

Renting the Floor

The Rational Choice

For an individual organization, renting foundational software is almost always the correct decision. Building is expensive, risky, and rarely justifies the investment.

The Crucial Difference

The algorithm is the same everywhere. The inputs differ.

Condition China India
Can you buy/rent it? No (sanctions, delete-IOE) Yes (open market)
Is it core? Yes (state banks, platforms) No (services firms)
Does it differentiate? Yes (Singles' Day scale) No
Is building cheaper? Yes (no alternative) No (renting is cheaper)
Strategic independence asked? Yes (state policy) No (market logic)

****Result:**** China built. India rented. The remainder of this essay explains why this matters.

The Puzzle

India has every input required to build foundational software—scale, talent, capital, and a large domestic market. Yet the output is near zero. The remainder of this essay explains why.
